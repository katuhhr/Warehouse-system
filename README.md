# Информационная система для склада автозапчастей

Интерфейс для управления складом автозапчастей, созданный на PyQt5.

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Настройте подключение к базе данных в файле `database.py`:
   - По умолчанию используется PostgreSQL
   - Если PostgreSQL недоступен, автоматически используется SQLite (файл `warehouse.db`)

3. Запустите приложение:
```bash
python main.py
```

## Структура проекта

- `main.py` - главный файл приложения
- `database.py` - модуль для работы с базой данных
- `login_window.py` - окно авторизации
- `main_window.py` - главное окно
- `products_window.py` - окно списка товаров
- `add_product_window.py` - форма добавления/редактирования товара
- `brand_window.py` - окно управления марками автомобилей
- `model_window.py` - окно управления моделями автомобилей
- `category_window.py` - окно управления категориями
- `clients_window.py` - окно списка клиентов
- `client_form_window.py` - форма добавления/редактирования клиента
- `orders_window.py` - окно списка заказов
- `order_form_window.py` - форма создания/редактирования заказа

## Функциональность

- Авторизация пользователя
- Управление товарами (добавление, редактирование, удаление, поиск)
- Управление клиентами
- Управление заказами
- Управление справочниками (марки, модели, категории)

## База данных

Приложение поддерживает PostgreSQL и SQLite. Структура БД соответствует ER-диаграмме:
- carbrand (марки автомобилей)
- carmodel (модели автомобилей)
- category (категории товаров)
- Product (товары)
- Client (клиенты)
- Order (заказы)
- Receiving (поступления)
- StorageCell (ячейки склада)
